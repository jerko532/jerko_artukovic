Vježba 10 i 11 – Web stranica i GitHub

U ovoj vježbi naučio sam kako izraditi HTML dokument koji sadrži multimedijalni sadržaj poput teksta, slika, zvuka, videa i linkova. Stilizirao sam stranicu pomoću CSS-a, naučio kreirati klase, navigacijske liste i umetnuti sadržaje poput YouTube videa i zvučnih isječaka.

Osim izrade HTML i CSS strukture, u vježbi 11 naučio sam kako napraviti besplatni račun na GitHubu i postaviti cijelu web stranicu online koristeći GitHub Pages. Time sam omogućio javnu dostupnost svih mojih vježbi i projekata.
